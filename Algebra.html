Algebra Formula Sheet

Algebraic Identities

  • $(a+b)^2 = a^2 + 2ab + b^2$
  • $(a-b)^2 = a^2 - 2ab + b^2$
  • $a^2 - b^2 = (a-b)(a+b)$
  • $(x+y+z)^2 = x^2 + y^2 + z^2 + 2(xy + yz + zx)$
  • $(x+y)^3 = x^3 + y^3 + 3xy(x+y)$
  • $(x-y)^3 = x^3 - y^3 - 3xy(x-y)$

Factorization Formulas

  • $x^3 + y^3 = (x+y)(x^2 - xy + y^2)$
  • $x^3 - y^3 = (x-y)(x^2 + xy + y^2)$
  • $x^n - y^n = (x-y)(x^{n-1} + x^{n-2}y + \dots + xy^{n-2} + y^{n-1})$
  • $x^n + y^n = (x+y)(x^{n-1} - x^{n-2}y + \dots - xy^{n-2} + y^{n-1})$ (n odd)

Binomial Theorem

  • $(a+b)^n = \displaystyle\sum_{k=0}^{n} \binom{n}{k} a^{n-k}b^k$
  • Pascal's Triangle gives coefficients for binomial expansion.

Arithmetic Progression (AP)

  • $n$th term: $a_n = a + (n-1)d$
  • Sum of $n$ terms: $S_n = \frac{n}{2}[2a + (n-1)d]$

Geometric Progression (GP)

  • $n$th term: $a_n = ar^{n-1}$
  • Sum of $n$ terms: $S_n = a \frac{r^n - 1}{r-1}, r \neq 1$
  • Sum of infinite GP (|r|<1): $S_\infty = \frac{a}{1-r}$

Logarithms

  • $\log(ab) = \log a + \log b$
  • $\log(\frac{a}{b}) = \log a - \log b$
  • $\log(a^n) = n \log a$
  • $\log_a b = \frac{\log b}{\log a}$

Miscellaneous Formulas

  • Sum of first $n$ natural numbers: $1 + 2 + \dots + n = \frac{n(n+1)}{2}$
  • Sum of squares: $1^2 + 2^2 + \dots + n^2 = \frac{n(n+1)(2n+1)}{6}$
  • Sum of cubes: $1^3 + 2^3 + \dots + n^3 = \left(\frac{n(n+1)}{2}\right)^2$
