Algebra Formulas (up to Class 12)

Basic Algebraic Identities

  • (a + b)² = a² + 2ab + b²
  • (a - b)² = a² - 2ab + b²
  • a² - b² = (a - b)(a + b)
  • (x + a)(x + b) = x² + (a + b)x + ab
  • (x - a)(x - b) = x² - (a + b)x + ab

Factorization Formulas

  • Sum of cubes: a³ + b³ = (a + b)(a² - ab + b²)
  • Difference of cubes: a³ - b³ = (a - b)(a² + ab + b²)
  • Quadratic: ax² + bx + c = a(x - α)(x - β), α & β = roots

Sequences & Progressions

  • Sum of first n natural numbers: 1 + 2 + ... + n = n(n + 1)/2
  • Sum of squares: 1² + 2² + ... + n² = n(n + 1)(2n + 1)/6
  • Sum of cubes: 1³ + 2³ + ... + n³ = [n(n + 1)/2]²
  • Arithmetic Progression (AP): Sₙ = n/2 [2a + (n-1)d]
  • Geometric Progression (GP): Sₙ = a(1 - rⁿ)/(1 - r), r ≠ 1

Quadratic Equations

  • ax² + bx + c = 0
  • Roots: x = [-b ± √(b² - 4ac)] / 2a
  • Sum of roots: α + β = -b/a
  • Product of roots: αβ = c/a

Miscellaneous Formulas

  • Binomial Theorem: (a + b)ⁿ = Σ [nCk aⁿ⁻ᵏ bᵏ], k = 0 to n
  • Logarithmic properties: log(ab) = log a + log b, log(a/b) = log a - log b
  • Exponential rules: a^m × a^n = a^(m+n), (a^m)^n = a^(mn)
