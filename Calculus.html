Calculus Formula Sheet

Limits

  • $\lim_{x \to a} c = c$
  • $\lim_{x \to a} x = a$
  • $\lim_{x \to a} [f(x) \pm g(x)] = \lim_{x \to a} f(x) \pm \lim_{x \to a} g(x)$
  • $\lim_{x \to a} [f(x) \cdot g(x)] = (\lim_{x \to a} f(x)) (\lim_{x \to a} g(x))$
  • $\lim_{x \to a} \frac{f(x)}{g(x)} = \frac{\lim_{x \to a} f(x)}{\lim_{x \to a} g(x)}, \text{if } \lim_{x \to a} g(x) \neq 0$

Derivatives

  • $(c)' = 0$
  • $(x^n)' = n x^{n-1}$
  • $(\sin x)' = \cos x$
  • $(\cos x)' = -\sin x$
  • $(\tan x)' = \sec^2 x$
  • $(\cot x)' = -\csc^2 x$
  • $(\sec x)' = \sec x \tan x$
  • $(\csc x)' = -\csc x \cot x$
  • $(e^x)' = e^x$
  • $(a^x)' = a^x \ln a$
  • $(\ln x)' = \frac{1}{x}$
  • $(\log_a x)' = \frac{1}{x \ln a}$

Rules of Differentiation

  • Sum Rule: $(f + g)' = f' + g'$
  • Product Rule: $(fg)' = f'g + fg'$
  • Quotient Rule: $\left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2}$
  • Chain Rule: $(f(g(x)))' = f'(g(x)) \cdot g'(x)$

Integrals

  • $\int c \, dx = cx + C$
  • $\int x^n \, dx = \frac{x^{n+1}}{n+1} + C, \, n \neq -1$
  • $\int \frac{dx}{x} = \ln |x| + C$
  • $\int e^x \, dx = e^x + C$
  • $\int a^x \, dx = \frac{a^x}{\ln a} + C$
  • $\int \sin x \, dx = -\cos x + C$
  • $\int \cos x \, dx = \sin x + C$
  • $\int \sec^2 x \, dx = \tan x + C$
  • $\int \csc^2 x \, dx = -\cot x + C$
  • $\int \sec x \tan x \, dx = \sec x + C$
  • $\int \csc x \cot x \, dx = -\csc x + C$

Miscellaneous Formulas

  • Definite integral property: $\int_a^b f(x) dx = F(b) - F(a)$
  • Linearity: $\int (af(x) + bg(x)) dx = a \int f(x) dx + b \int g(x) dx$
  • Area under curve: $A = \int_a^b f(x) dx$
