Calculus Formulas (up to Class 12)

Limits

  • lim (x→a) [c] = c
  • lim (x→a) [x] = a
  • lim (x→a) [xⁿ] = aⁿ
  • lim (x→0) (sin x)/x = 1
  • lim (x→0) (1 - cos x)/x = 0

Derivatives

  • d/dx [c] = 0
  • d/dx [xⁿ] = n·xⁿ⁻¹
  • d/dx [sin x] = cos x
  • d/dx [cos x] = -sin x
  • d/dx [tan x] = sec² x
  • Product rule: d/dx [u·v] = u'v + uv'
  • Quotient rule: d/dx [u/v] = (u'v - uv') / v²
  • Chain rule: d/dx [f(g(x))] = f'(g(x))·g'(x)

Integrals

  • ∫ c dx = cx + C
  • ∫ xⁿ dx = xⁿ⁺¹ / (n+1) + C, n ≠ -1
  • ∫ eˣ dx = eˣ + C
  • ∫ sin x dx = -cos x + C
  • ∫ cos x dx = sin x + C
  • ∫ sec² x dx = tan x + C
  • ∫ csc² x dx = -cot x + C
  • ∫ sec x tan x dx = sec x + C
  • ∫ csc x cot x dx = -csc x + C
