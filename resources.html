Notes / How to use

short tips
• Formula sheets are topic-focused — use them for quick revision before tests.
• Worksheets are organized by topic (not by class). Pick practice sheets by subject (Algebra, Trigonometry, Calculus, ...).
• If you need printable PDFs, open the topic page and use browser Print → Save as PDF.
• If a link shows 404, check that the file name is exactly as listed here (case-sensitive on some hosting). File names must match: Algebra.html, algebra.html, etc.
