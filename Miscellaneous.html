Miscellaneous Formulas (up to Class 12)

Algebraic Identities

  • (a + b)² = a² + 2ab + b²
  • (a − b)² = a² − 2ab + b²
  • a² − b² = (a − b)(a + b)
  • (x + a)(x + b) = x² + (a+b)x + ab
  • (a + b + c)² = a² + b² + c² + 2(ab + bc + ca)
  • Cube formula: a³ + b³ = (a+b)(a² − ab + b²)
  • Cube formula: a³ − b³ = (a−b)(a² + ab + b²)

Binomial Theorem

  • (a + b)ⁿ = Σ (nCr) aⁿ⁻ʳ bʳ , r = 0 to n
  • nCr = n! / [r!(n−r)!]
  • Sum of coefficients: 2ⁿ
  • Alternating sum of coefficients: 0 (for n ≥ 1)

Logarithmic Properties

  • log(a·b) = log a + log b
  • log(a/b) = log a − log b
  • log(aⁿ) = n·log a
  • logₐ a = 1
  • logₐ 1 = 0

Exponents / Powers

  • aᵐ · aⁿ = aᵐ⁺ⁿ
  • aᵐ / aⁿ = aᵐ⁻ⁿ
  • (aᵐ)ⁿ = aᵐⁿ
  • (a·b)ⁿ = aⁿ · bⁿ
  • (a/b)ⁿ = aⁿ / bⁿ

Series / Progressions

  • Arithmetic Progression (AP): aₙ = a₁ + (n−1)d
  • Sum of n terms in AP: Sₙ = n/2 [2a₁ + (n−1)d]
  • Geometric Progression (GP): aₙ = a₁ · rⁿ⁻¹
  • Sum of n terms in GP: Sₙ = a₁ (rⁿ−1)/(r−1), r ≠ 1

Matrices & Determinants

  • Determinant of 2×2 matrix: |A| = ad − bc
  • Determinant of 3×3 matrix: |A| = a(ei−fh) − b(di−fg) + c(dh−eg)
  • Matrix addition: (A + B)ᵢⱼ = Aᵢⱼ + Bᵢⱼ
  • Matrix multiplication: (AB)ᵢⱼ = Σ Aᵢₖ · Bₖⱼ
