Note: Use standard maths format. Variables like x, y are supported. For nth root, type nthroot(a,n) (e.g., nthroot(8,3) → 2). Trigonometry, logarithms, and exponents work without brackets (e.g., sin30, log100). If the input is invalid, the solver will return Invalid Input.
