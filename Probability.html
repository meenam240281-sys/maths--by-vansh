Probability & Statistics Formulas (up to Class 12)

Probability

  • P(E) = Number of favorable outcomes / Total outcomes
  • P(A') = 1 - P(A) (Complement rule)
  • P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
  • P(A ∩ B) = P(A) × P(B) (Independent events)
  • P(A|B) = P(A ∩ B) / P(B) (Conditional probability)

Statistics

  • Mean: x̄ = (Σxᵢ) / n
  • Median: Middle value after arranging data
  • Mode: Value that occurs most frequently
  • Variance (population): σ² = Σ(xᵢ - μ)² / n
  • Variance (sample): s² = Σ(xᵢ - x̄)² / (n-1)
  • Standard deviation: σ = √Variance
  • Probability distribution: P(x) for discrete random variable

Combinations & Permutations

  • nPr = n! / (n-r)! (Permutation of r objects from n)
  • nCr = n! / [r!(n-r)!] (Combination of r objects from n)
  • nPr = nCr × r!
