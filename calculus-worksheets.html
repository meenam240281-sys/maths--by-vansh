Calculus Practice Worksheets

Limits

  1. Find: \(\lim_{x \to 0} \frac{\sin x}{x}\)
  2. Evaluate: \(\lim_{x \to \infty} \frac{2x^2 + 3}{x^2 - 5}\)
  3. Find: \(\lim_{x \to 1} \frac{x^2 - 1}{x - 1}\)
  4. Evaluate: \(\lim_{x \to 0} \frac{1 - \cos x}{x^2}\)

Derivatives

  1. Differentiate: \(y = x^3 + 5x^2 - 7x + 2\)
  2. Find \(\frac{dy}{dx}\) if \(y = \sin x \cdot \cos x\)
  3. Differentiate: \(y = \ln x\)
  4. Find slope of the tangent to \(y = x^2\) at \(x = 3\)

Applications of Derivatives

  1. Find the maximum and minimum values of \(y = x^2 - 4x + 3\).
  2. The cost function is \(C(x) = 5x + 100\). Find the marginal cost when \(x = 20\).
  3. A particle moves such that \(s = t^3 - 6t^2 + 9t\). Find velocity and acceleration at \(t=2\).
  4. Find the equation of tangent to \(y = x^2\) at point (2,4).

Integration

  1. Evaluate: \(\int (3x^2 + 2x + 1) \, dx\)
  2. Find: \(\int \sin x \, dx\)
  3. Evaluate: \(\int \frac{1}{x} \, dx\)
  4. Find: \(\int e^x \, dx\)

Definite Integrals

  1. Evaluate: \(\int_0^1 x^2 \, dx\)
  2. Evaluate: \(\int_0^{\pi/2} \sin x \, dx\)
  3. Find: \(\int_1^e \frac{1}{x} \, dx\)
  4. Evaluate: \(\int_0^2 (2x+1) \, dx\)
